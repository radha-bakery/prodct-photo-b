# Product Photo Tool Pro - Movable Product Layer

Features:
- White background fixed/stable
- Only removed-background product layer moves
- Drag product with mouse/finger on canvas
- Rotate, flip, zoom, center
- Manual erase/restore brush
- Grid/safe margin
- 400×400 final JPG
- Compress target under 190 KB
- Custom filename
- GitHub Pages compatible
- No backend / no API key

Upload these files to GitHub Pages:
- index.html
- style.css
- script.js
- README.md


## Safe Restore Version
If AI removes too much product:
1. Select `Restore Cut Part`.
2. Brush on the missing product area.
3. Use brush size slider for bigger/smaller repair.
4. Use Erase only for extra background.


## Drag Restore Path
Restore and Erase now work as continuous drag strokes:
- Mouse/finger press and drag from start to end.
- The full path is restored/erased, not only single tap points.
- Undo removes the whole last dragged stroke.
