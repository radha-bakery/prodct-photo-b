# Product Photo Tool Pro - Movable Product Layer

Features:
- White background fixed/stable
- Only removed-background product layer moves
- Drag product with mouse/finger on canvas
- Rotate, flip, zoom, center
- Manual erase/restore brush
- Grid/safe margin
- 400×400 final JPG
- Compress target under 190 KB
- Custom filename
- GitHub Pages compatible
- No backend / no API key

- index.html
- style.css
- script.js
