# Product Photo BG Remove Tool

GitHub Pages compatible static website.

Features:
- Background removal in browser using @imgly/background-removal
- Automatic white background
- 400×400 px output
- JPG only download
- Compresses target under 190 KB
- No remove.bg API key
- No backend

## Upload on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files:
   - index.html
   - style.css
   - script.js
3. Go to Settings → Pages.
4. Source: Deploy from branch.
5. Branch: main / root.
6. Open the GitHub Pages link.

Note:
First processing may take time because the AI model downloads in the browser. Internet is required.
Best browser: Chrome or Edge.
